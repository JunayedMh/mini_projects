**The user enters two groups of items** (e.g., two shopping lists),
and the program performs set operations on them.

## Rules:

- User enters items for Group A (comma-separated).
- User enters items for Group B (comma-separated).
- Convert both inputs to sets (lowercase, stripped of spaces).
- Menu:
1. Union (all unique items from both)
2. Intersection (items in BOTH groups)
3. Difference A-B (in A but NOT in B)
4. Difference B-A (in B but NOT in A)
5. Symmetric Difference (in either but NOT both)
6. Check if a specific item exists in A, B, or both
7. Exit
- Display results sorted alphabetically.
